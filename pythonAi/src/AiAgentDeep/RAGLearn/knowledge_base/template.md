# 教育背景模板（适用于简历）

## 写作原则（必须遵守）

1. **顺序倒序**：最新学历放最上面。
2. **简洁有力**：每条不超过 2-3 行，重点突出与目标职位相关的亮点。
3. **量化 & 关键词**：GPA/排名/奖学金/核心课程/项目/实习 用数字量化；融入职位JD关键词（如“机器学习”“数据结构”“项目管理”）。
4. **STAR 思路融入**：如果有相关项目/竞赛/社团经历，用 STAR 框架精简描述（Situation/Task → Action → Result）。
5. **格式规范**：
   - 学校名称 + 学位 + 专业 + 时间
   - GPA/排名（如果优秀才写）
   - 核心课程（3-5 门相关课程）
   - 荣誉/奖项
   - 相关经历（项目/实习/研究/社团，用 STAR 精炼）
